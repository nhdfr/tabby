# Installation

## Prerequisites

- Go 1.21 or higher


## From AUR (Arch Linux)

If you're on Arch Linux or a derivative, you can install Tabby from the AUR using yay:

```bash
yay -S tabby-cli
```

## Using Go Install

The easiest way to install Tabby on any platform with Go is:

```bash
go install github.com/nhdfr/tabby@latest
```

Make sure your Go bin directory is in your PATH:

```bash
export PATH=$PATH:$(go env GOPATH)/bin
```


## Building from Source

Clone the repository and build:

```bash
git clone https://github.com/nhdfr/tabby.git
cd tabby
go build -o tabby
```

## Verify Installation

Check that Tabby is installed correctly:

```bash
tabby --help
```

You should see the help output with available commands.
